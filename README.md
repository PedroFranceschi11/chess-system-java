# ♟️ Sistema de Xadrez

Projeto desenvolvido em Java durante o curso do Nélio Alves com o objetivo de praticar os principais conceitos de Programação Orientada a Objetos.

## 📚 Conceitos aplicados

- Programação Orientada a Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Classes abstratas
- Tratamento de exceções
- Coleções
- Organização em camadas

## 🎮 Funcionalidades

- Movimentação das peças
- Validação das jogadas
- Captura de peças
- Controle de turnos
- Xeque
- Xeque-mate
- Promoção de peões

## 🛠️ Tecnologias

- Java
- Git
- GitHub

## 🎯 Objetivo

Este projeto foi desenvolvido para consolidar os conceitos estudados durante minha formação em Java, com foco em Programação Orientada a Objetos e boas práticas de organização de código.
